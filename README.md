# ts-tic-tac-toe
This is game tic-tac-toe in typescript

## Test app
[tic-tac-toe](https://sebastiangolian.github.io/ts-tic-tac-toe/)

## Getting started
```bash
npm install
```
Build project
```bash
npm run build
```

## Scripts
Watch project
```bash
npm run watch
```
Clear temporary files
```bash
npm run clear
```
Deploy files in gh-pages
```bash
npm run deploy
```
